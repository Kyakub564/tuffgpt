## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Kyakub564/tuffgpt.git
   ```
2. Navigate into the project directory:
   ```bash
   cd tuffgpt
   ```
3. Install the required dependencies:
